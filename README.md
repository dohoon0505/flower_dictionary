# 꽃배달 창업 백과사전 · Flower Startup Encyclopedia

> 온라인 꽃 배달 쇼핑몰 상담사를 위한 실전 응대 핸드북.
> **현재 버전 v1.1.0**

---

## 이 프로젝트는

온라인 꽃 배달 쇼핑몰을 운영하면서 직접 고객 상담을 책임지는 분을 위한 가이드북입니다. 시장 분석·창업 절차가 아니라 **"고객 문의가 들어왔을 때 어떻게 응대할 것인가"** 에 초점이 있습니다.

| 대상 | 진입점 | 특징 |
| --- | --- | --- |
| 사람 | `index.html` | 브라우저에서 열면 바로 가이드북 |
| AI 에이전트 | [`AGENTS.md`](AGENTS.md) + [`system.json`](system.json) | 1–2 파일 read로 모든 쿼리 해결 |
| 기여자 | [`CLAUDE.md`](CLAUDE.md) | 3계층 작업 원칙 · 수정 워크플로 |

## 주요 기능

- **GitBook식 통합 검색** — 상단 헤더의 `검색` 버튼 또는 단축키 `Cmd/Ctrl+K` / `/` 로 호출. 챕터·아티클·본문 텍스트까지 한 번에 찾아 바로 이동.
- **라이트 / 다크 모드** — 토큰 기반 컬러 스왑.
- **큰글씨 모드** — `Aa+` 토글로 폰트·여백·둥글기 일괄 1.2배 확대.
- **모바일 반응형** — 360px viewport에서 가로 스크롤 0건 검증.
- **선호 영속화** — 테마·큰글씨 설정은 `localStorage` 에 저장됨.

## 폴더 구조

```
flower_dictionary/
├── index.html                  ← 운영 진본 (단일 진입점)
├── assets/
│   ├── css/main.css
│   └── js/main.js
│
├── system.json                 ← 루트 매니페스트 (chapters + releases)
├── analyses/                   ← 챕터 콘텐츠 데이터
│   └── {chapter-id}/chapter.json
│
├── scripts/validate.mjs        ← 무결성 검증
├── AGENTS.md
├── CLAUDE.md
└── README.md
```
